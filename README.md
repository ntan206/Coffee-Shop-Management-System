# Coffee-Shop-Management-System

**DỰ ÁN HỆ THỐNG QUẢN LÝ BÁN CÀ PHÊ - TEAM JESPIRE**

**Chức năng hệ thống:**
> Nhân Viên:
- Nhận Order khách hàng, chỉnh số lượng đường, đá và thêm vào cart
- Tạo tài khoản thành viên cho khách hàng cần nhu cầu tạo thành viên
- Hỗ trợ thanh toán (có áp mã Khuyến Mãi, mã Thành Viên) và thanh toán qua QR,Tiền Mặt
- Tự động truy xuất Database để lấy dữ liệu Menu, và dữ liệu Products
- Giao diện hệ thống trực quan, dễ sử dụng
> Quản Lý:
- Quản Lý Doanh Thu
- Quản Lý Hóa Đơn
- Quản Lý Đơn Hàng
- Quản Lý User
- Quản Lý Tài Khoản Thành Viên
- Quản Lý Khuyến Mãi

**Ngôn Ngữ và Add-on sử dụng trong khi làm dự án:**
- Java (Ngôn Ngữ Lập Trình - Thiết Kế Front End và Back - End)
- SQL (Ngôn Ngữ Truy Vấn - SQL Server hoặc MYSQL Server)
- Add-on khác như PDF, JDBC, MYSQL JDBC, Flatfile.
